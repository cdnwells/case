# Instructions

## Target Machine

All commands MUST be executed on the remote desktop at `192.168.0.18` via SSH. Do NOT run commands on the local worker environment. Always use `ssh 192.168.0.18` (or `ssh cdnwell@192.168.0.18`) to execute commands on the target machine.

When the user says "the computer" or asks you to do something, it always refers to `192.168.0.18`, not this machine.

Example:
- User: "파일 목록 보여줘" → Run `ssh 192.168.0.18 'ls -la'`
- User: "git 상태 확인해줘" → Run `ssh 192.168.0.18 'cd /path && git status'`
